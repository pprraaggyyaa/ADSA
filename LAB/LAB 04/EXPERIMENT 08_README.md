# Longest Common Subsequence (LCS)

## Aim
To find the **Longest Common Subsequence (LCS)** between two given strings
using **Dynamic Programming** and to display all possible common subsequences.

## Description
This program takes two input strings and computes the length of their
Longest Common Subsequence using a dynamic programming approach.

A DP table is constructed to store intermediate results, which helps in:
- Finding the length of the LCS
- Reconstructing the LCS string

Additionally, the program uses recursion to print all possible common
subsequences between the two strings.

## Algorithm Used
- Dynamic Programming
- Recursion

## Input
- First string
- Second string

## Output
- Length of the Longest Common Subsequence
- The LCS string
- All possible common subsequences
