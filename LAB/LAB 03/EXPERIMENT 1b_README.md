# Comparison of Sorting Algorithms

## Aim
To implement and compare different **sorting algorithms**—**Quick Sort**,
**Merge Sort**, and **Heap Sort**—using a menu-driven program.

## Description
This program allows the user to sort an array using one of the following
sorting techniques:
1. Quick Sort
2. Merge Sort
3. Heap Sort

Based on the user’s choice, the corresponding sorting algorithm is applied
to the input array and the sorted result is displayed.

## Algorithms Used
- Quick Sort
- Merge Sort
- Heap Sort

## Input
- Number of elements `n`
- `n` array elements
- Choice of sorting algorithm

## Output
- Sorted array using the selected sorting algorithm

## Applications
