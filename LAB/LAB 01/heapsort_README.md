## Aim
To implement the **Heap Sort algorithm** to sort a given list of elements
in ascending order.

## Description
This program implements the Heap Sort technique using a **max heap**.
The algorithm first builds a max heap from the input array and then repeatedly
extracts the maximum element and places it at the end of the array.

The heap property is maintained using the `heapify` function, ensuring efficient
sorting.

## Algorithm Used
- Heap Sort (using Max Heap)

## Input
- Number of elements `n`
- `n` integer elements

## Output
- The sorted array in ascending order
